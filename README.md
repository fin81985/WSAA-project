# Expense Tracker Web Application

## Description
This project is a Flask-based web application that allows users to manage their expenses.

The application supports:
- Create expenses
- View expenses
- Update expenses
- Delete expenses

It uses a RESTful API connected to a SQLite database.
The frontend communicates with the backend using JavaScript (AJAX).

---

## Technologies Used
- Python (Flask)
- SQLite
- HTML, CSS, JavaScript

---

## How to Run

1. Install dependencies:
pip install -r requirements.txt

2. Run the application:
python app.py

3. Open in browser:
http://127.0.0.1:5000/

---

## API Endpoints

GET /expenses  
POST /expenses  
PUT /expenses/<id>  
DELETE /expenses/<id>

---

## Features
- Full CRUD functionality
- Dynamic frontend using JavaScript
- Clean project structure
